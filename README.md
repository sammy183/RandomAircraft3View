# How to use:
1. Download the exe
2. Run
3. Be curious! 

With the json in the same folder, the exe will immediately open a random 3-view from http://richard.ferriere.free.fr/3vues/3vues.html,
otherwise it'll gather all the links (takes around 30s) and make a json in the folder with the exe. From there it won't gather them again unless you specify "RandomAircraft3View.exe --gather-links" in the command line.

# Why did I make this?
First and foremost, I love discovering new aircraft. I've also found examining 3-views to be a great way to learn about aircraft design from both a historical and engineering perspective.
Just by looking and asking questions like, "Why are the wings that shape? Why does it have that number of engines? How did they choose the tail dihedral/placement?", I've learned an incredible amount.
As an aircraft history buff too, it's lead me down some fascinating rabbit holes! I hope it does the same for you.

This project is simply a wrapper of Richard Ferriere's site. He deserves all the credit for uploading so many 3-views of such a diverse range of aircraft.
Please, check out his other work at http://richard.ferriere.free.fr/index.html!
